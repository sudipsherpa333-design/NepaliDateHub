## 🌟 Overview

**CalcHub Nepal** is a modern, fast, and highly interactive web application designed to simplify complex financial and utility calculations for Nepali citizens and businesses. Built with a focus on user experience, it features smooth animations, dark mode support, and precise localized formatting (Nepali Rupee `रु`, Lakhs/Crores numbering system).

Whether you are calculating your next home loan, figuring out your income tax liability based on the latest government slabs, or just need a quick GST/VAT extraction, CalcHub Nepal provides a seamless, app-like experience right in your browser.

<br />

## ✨ Core Features

| Feature | Description |
| :--- | :--- |
| **🏦 EMI Calculator** | Calculate Equated Monthly Installments with detailed amortization schedules and interactive pie charts. |
| **🏛️ Income Tax** | Up-to-date tax bracket calculations for individuals and couples based on the latest Nepal Government fiscal policies. |
| **📊 GST / VAT** | Quickly add or extract Value Added Tax (VAT) for goods and services with a single click. |
| **📅 Age Calculator** | Precise age calculation down to months and days, perfect for official documentation. |
| **🌓 Dark Mode** | First-class dark mode support for comfortable viewing in any lighting condition. |
| **📱 Responsive** | Beautifully crafted interface that works seamlessly on desktop, tablet, and mobile devices. |

<br />

## 🛠️ Tech Stack & Architecture

CalcHub Nepal is built on a modern frontend stack, prioritizing performance, type safety, and developer experience.

- **Frontend Framework**: [React 18](https://react.dev/) (Functional Components & Hooks)
- **Language**: [TypeScript](https://www.typescriptlang.org/) (Strict Mode Enabled)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Utility-First, Custom Theme Configuration)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) (Spring Physics, Layout Animations)
- **Data Visualization**: [Recharts](https://recharts.org/) (Responsive SVG Charts)
- **Icons**: [Lucide React](https://lucide.dev/) (Consistent, Crisp SVGs)
- **Build Tool**: [Vite](https://vitejs.dev/) (Lightning Fast HMR & Optimized Builds)

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Ensure you have the following installed:
- Node.js (v18 or higher recommended)
- npm, yarn, or pnpm

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/calchub-nepal.git
   cd calchub-nepal
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or yarn install
   # or pnpm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   # or yarn dev
   # or pnpm dev
   ```

4. **View the app:**
   Open your browser and navigate to `http://localhost:3000`.
